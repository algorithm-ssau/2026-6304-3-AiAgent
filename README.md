# ИИ-агент для техподдержки

**ИИ-агент техподдержки** — это веб-приложение, которое автоматически анализирует текстовые обращения пользователей и помогает операторам быстро реагировать.

При отправке сообщения агент определяет:
- **Категорию**: доступ (access), ошибка (bug), оплата (payment), предложение (suggestion), другое (other)
- **Приоритет**: высокий (high) — сервис не работает, средний (medium) — мешает, есть обход, низкий (low) — вопрос или предложение
- **Пояснение**: краткое обоснование выбора категории и приоритета
- **Готовый ответ оператору**: вежливый, развёрнутый ответ (3–4 предложения) с рекомендациями, который можно скопировать одним нажатием

Все обращения сохраняются в историю (CSV-файл). На отдельной странице доступна статистика с графиками распределения по категориям и приоритетам. Также можно скачать отчёт в ZIP-архиве, содержащий CSV-лог и оба графика в формате PNG.

Продукт предназначен для малых и средних IT-компаний, интернет-магазинов, служб поддержки — любых организаций с большим потоком однотипных обращений. Он сокращает время реакции на критичные проблемы и снижает нагрузку на живых операторов.

## Запуск проекта

### Требования
- Python 3.10 или новее
- Git
- Учётная запись в [GigaChat API](https://developers.sber.ru) (Client ID и Authorization key)

### Инструкция

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/algorithm-ssau/2026-6304-3-AiAgent.git
   cd 2026-6304-3-AiAgent
2. **Создайте и активируйте виртуальное окружение**
   ```bash
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   # macOS/Linux:
   source venv/bin/activate
3. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
4. **Настройте API-ключи**
   Создайте в корне проекта файл .env и добавьте в него:
   GIGACHAT_CLIENT_ID=ваш_client_id
   GIGACHAT_AUTH_KEY=ваш_authorization_key
   Ключи можно получить в личном кабинете GigaChat.
5. **Запустите сервер**
   ```bash
   uvicorn main:app --reload
6. **Откройте веб-интерфейс**
   Перейдите в браузере по адресу http://127.0.0.1:8000

Готово! Теперь вы можете отправлять обращения и пользоваться всеми функциями: историей, статистикой, копированием готовых ответов.
