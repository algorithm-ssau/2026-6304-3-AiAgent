# 2026-6304-3-Agent
# ИИ-агент для категоризации обращений техподдержки

## Что делает продукт

Веб-приложение, которое автоматически классифицирует обращения пользователей в техподдержку.  
На основе текста обращения ИИ-агент определяет:

- **Категорию**: доступ (access), ошибка (bug), оплата (payment), предложение (suggestion), другое (other)
- **Приоритет**: высокий (high), средний (medium), низкий (low)
- **Пояснение**: краткое обоснование решения

Программа сохраняет историю обращений в CSV-файл и предоставляет страницу для просмотра логов.

## Как запустить

- Python 3.10 или выше
- Git
- API-ключ YandexGPT (получите в Yandex Cloud)

## Архитектура решения

Проект состоит из следующих компонентов:

- **Frontend**: HTML + CSS + JavaScript (одностраничное приложение с формой и страницей истории)
- **Backend**: FastAPI (Python) — обрабатывает запросы, вызывает LLM, ведёт логи
- **LLM**: YandexGPT (облачное API) — классифицирует обращения по промпту
- **Логирование**: CSV-файл `requests_log.csv` — хранит все запросы и ответы
- **Обработка ошибок**: fallback-ответы при недоступности API, таймаутах, невалидном JSON

**Схема взаимодействия:**
Пользователь → Веб-форма → FastAPI (endpoint /classify) → YandexGPT API →
получение JSON → логирование в CSV → ответ пользователю
